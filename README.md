Follow these steps to set up Repo locally:

1. Clone the repository: 
```

```

2. Install the required dependencies:
```
cd multi-select-chip-component
npm install
```


3. Start the server:
```
npm run dev
```

5. Access server in your web browser at `http://localhost:3000`.
